# P3. Training

1. 스크립트 작성 시 적용 공통 규칙
 1)기존 old_scripts 폴더 내 스크립트를 기반
 2)신규 스크립트 실행후 log 기록 생성되도록 작성
 3)log 기록 최소 요건 : 에러발생, 누락정보, 기본적으로 수행한 내용의 통계 자료, 해당 스크립트 검토 후 중요 기록 사항을 제시한 후 log 기록에 반영
 4)수행 결과물 검증 방법 제안 및 확보
 5)현재 구축된 컴퓨팅 환경 활용 극대화
     CPU: AMD Ryzen Threadripper PRO 7975WX 32-Cores     (4.00 GHz)
     RAM : 128 GB
     GPU : NVIDIA RTX 4000 Ada VRAM 20 GB X 2개
 6)GPU 처리 방식은 old script 폴더내 p3_cv_gpu_trainer_fast_v3.py 규칙 따름
 7)모든 코드에 정규화 코드 반영여부 확인
 8) scaffold mask를 평가 모드별로 on/off 할 수 있도록 반영
 9) 지금부터 P3 관련 코드 수정은 script 폴더 내 파일만 대상으로 수행 (old_scripts 수정 금지)
 10)학습/샘플링/검증 전 과정에서 데이터 단위는 반드시 pair_id 기준으로 관리
 11)모든 학습 입력 데이터는 pair_id별로 반드시 양성 1개 + 음성 10개(총 11개) 구성 유지
 12)subset 생성 시 row 단위 샘플링 금지, pair_id 단위 샘플링으로 11개 세트 유지
 13)배치 실행 및 학습 시작 전 pair_id 규칙(11/1/10, target 일관성, cv_fold 일관성) 검증 필수
